## 🛠️ Setup Your Environment

### Install Python
```bash
# Windows
https://www.python.org/downloads/

# Mac
brew install python3

# Linux
sudo apt-get install python3
```

### Create Virtual Environment
```bash
# Create
python -m venv venv

# Activate
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate
```

### Install Required Packages
```bash
pip install requests numpy pandas matplotlib django
```

## 🐛 Debugging Help

### If you get an error:

1. **Read the error message carefully**
   ```
   FileNotFoundError: [Errno 2] No such file or directory
   ```
   → File path is wrong

2. **Check line number** indicated in error

3. **Look for typos**
   - Indentation
   - Variable names
   - Parentheses/brackets

4. **Use print() for debugging**
   ```python
   print(f"Variable value: {x}")
   ```

5. **Search on Stack Overflow** - Likely someone had same issue
